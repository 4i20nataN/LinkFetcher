## ✨ About

**LinkFetch** is a modern desktop application for downloading videos, audio, playlists, subtitles, and other media from hundreds of supported websites.

Built with a clean and intuitive interface, LinkFetch combines the power of **yt-dlp** with a polished desktop experience focused on speed, simplicity, and privacy.

---

## 🚀 Features

- 🎥 Download videos in the highest available quality
- 🎵 Extract audio (MP3, M4A, FLAC...)
- 📺 Playlist support
- 🌍 Hundreds of supported websites
- 📝 Subtitle downloads
- ⚡ Fast downloads
- 📂 Custom download folders
- 🔒 Privacy-focused (no login required)
- 🎨 Modern minimal interface
- 🖥 Cross-platform

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/linkfetch.git
```

Enter the project

```bash
cd linkfetch
```

Install dependencies

```bash
npm install
```

Create your environment file

```bash
.env.local
```

Add your Gemini API Key

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Start development

```bash
npm run dev
```
